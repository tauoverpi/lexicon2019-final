**Gladiator combat**
====================

Start of turn
-------------

- restore some stamina

Player options
--------------

- Player pick two options

1. Guard
    - reduce chance to be hit
    - 75% chance to face enemy
2. Attack

    - standard
        + increced chance to be hit
    - special(optinal)

3. Move
    - Up/Down/Left/Right
4. Observe
    - see tierdness(stamina)/wound(health points)

End of turn
-----------
 - Gladiator stats modifide
